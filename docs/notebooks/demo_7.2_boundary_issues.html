def plot_dist(X_samples: np.ndarray, bins=100, *args, **kwargs):
    fig, ax = plt.subplots()
    sns.distplot(
        X_samples, 
        bins=bins, 
        ax=ax, 
        kde_kws={
            'linewidth': 7, 
            'color': 'black',
        },
        hist_kws={
            'color': 'green'
        }
    )
    return fig, ax

CDF Estimation¶

For the CDF estimation, I will use the quantile method.

In [36]:
# number of quantiles
n_quantiles = 1_000
n_quantiles = max(1, min(n_quantiles, n_samples))

# calculate the reference values (support), [0, 1]
references = np.linspace(0, 1, n_quantiles, endpoint=True)

# calculate quantiles
quantiles = np.percentile(X, references * 100)

This also looks OK. But what happens if we find some points that are outside of our distribution? In other words, what happens if I have points that lay outside of my support?
